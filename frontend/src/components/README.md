# Components

This directory contains all React components for the application.

## Structure

- **auth/** - Authentication related components (Login, Signup)
- **chat/** - Learning Arena chat interface components
- **dashboard/** - Study Dashboard components
- **upload/** - Document Upload Center components
- **voice/** - Voice interaction UI components
- **common/** - Shared/reusable components (Button, Input, Modal, etc.)
- **layout/** - Layout components (Header, Footer, Sidebar)
