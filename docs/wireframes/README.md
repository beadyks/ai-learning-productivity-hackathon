# UI/UX Wireframes - Voice-First AI Learning Assistant

## Overview

This directory contains wireframe documentation for the 8 key screens of the Voice-First AI Learning Assistant. These wireframes emphasize voice-first interaction, multilingual support, and a clean, distraction-free learning experience.

## Design Principles

1. **Voice-First** - Large, prominent microphone button
2. **Minimal Distractions** - Clean, focused interface
3. **Multilingual** - Hindi/English/Hinglish support visible
4. **Mobile-Responsive** - Works on all devices
5. **Accessibility** - High contrast, clear typography

## 8 Key Screens

1. [Landing Page](01-landing-page.md) - Hero, pricing, features
2. [Authentication](02-authentication.md) - Login/Register
3. [Dashboard](03-dashboard.md) - Study progress overview
4. [Document Upload](04-document-upload.md) - Upload interface
5. [Voice Interface](05-voice-interface.md) - Main learning screen
6. [Chat Interface](06-chat-interface.md) - Conversation view
7. [Study Planner](07-study-planner.md) - Plan creation
8. [Settings](08-settings.md) - User preferences

## Tools for Creating Wireframes

### Recommended Tools
- **Figma** (https://figma.com) - Professional, collaborative
- **Excalidraw** (https://excalidraw.com) - Quick sketches
- **Balsamiq** - Rapid wireframing
- **Draw.io** - Free, simple

### Color Palette
- Primary: #4F46E5 (Indigo)
- Secondary: #10B981 (Green)
- Background: #F9FAFB (Light Gray)
- Text: #111827 (Dark Gray)
- Accent: #F59E0B (Amber)

### Typography
- Headings: Inter Bold
- Body: Inter Regular
- Code: Fira Code
