## Key Elements

### Welcome Message
- Personalized greeting
- Time-based (morning/afternoon/evening)
- Motivational message

### Stats Cards (2 cards)
1. **This Week**
   - Hours studied
   - Topics completed
   - Progress percentage
   
2. **Current Goal**
   - Goal name
   - Deadline countdown
   - Completion percentage

### Continue Learning Card
- Last topic studied
- Time since last session
- Progress bar
- Quick action buttons

### Today's Schedule
- Time-based task list
- Status indicators (âœ“ completed, â†’ in progress, pending)
- Click to start session

### Recent Documents
- 3 most recent uploads
- Upload date
- Quick access

### Quick Actions (Bottom)
- Upload Document (primary)
- Start Voice Session (prominent)
- Create Study Plan

## Interactions
- Click cards to expand details
- Hover effects on action buttons
- Smooth transitions
- Pull-to-refresh on mobile
